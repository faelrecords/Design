# Design

Editor visual client-side para arquivos `Design.MD`.

## Recursos

- Edição por formulário e Markdown.
- Preview responsivo em tempo real.
- Importação e exportação `.md`.
- Exemplos incluídos no projeto.
- Persistência local por navegador.
- Deploy automático no GitHub Pages.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
